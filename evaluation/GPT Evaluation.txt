3) Prompt Framework (Production-Ready)
MASTER PROMPT (Step 1–3: Strategy)
You are a senior global macro portfolio manager.

Input data:
1. Events (with impact scores)
2. Macro commentary
3. Market intelligence summaries

Tasks:

Step 1 — Extract key macro signals:
- Growth direction
- Inflation direction
- Interest rate trajectory
- Liquidity conditions
- Geopolitical risk level

Step 2 — Classify market regime:
- Choose one: Early-cycle, Mid-cycle, Late-cycle, Recession, Stagflation
- Assign risk bias (risk-on / neutral / risk-off)
- Estimate volatility level

Step 3 — Generate investment strategy:
- स्पष्ट positioning (no generic statements)
- Include:
  - Factor tilts
  - Asset class preferences
  - Key risks

Rules:
- Be specific and non-generic
- Use institutional language
- Do not output JSON
SECTOR ALLOCATION PROMPT
Based on the strategy and market regime:

1. Identify sectors:
   - Overweight (top 3–5)
   - Neutral
   - Underweight (avoid)

2. Justify each using:
   - Macro drivers
   - Current events
   - Rate sensitivity

Constraints:
- Avoid generic reasoning
- Explicitly link to macro signals

Output format:
Bullet points only (no JSON)
STOCK / ETF SELECTION PROMPT
Select stocks and ETFs for each overweight sector.

Rules:
- Use highly liquid US-listed securities
- Prefer sector leaders or ETFs
- Max 3 per sector
- Avoid redundancy

Output:
- Sector → tickers
- One-line rationale per ticker
PORTFOLIO CONSTRUCTION PROMPT
Construct a portfolio using selected assets.

Constraints:
- Max 6 positions
- Max 30% per position
- Include at least 1 hedge or defensive asset
- Avoid high correlation

Output:
- Ticker
- Weight
- Role (growth / hedge / cyclical)
RISK CRITIQUE PROMPT
Act as a risk manager.

Critique the portfolio:
- Concentration risks
- Macro mismatch
- Hidden correlations
- Tail risks (geopolitics, rates)

Then suggest improvements.
FINAL JSON STRUCTURING PROMPT
Convert the following into structured JSON:

{
  strategy,
  sectors,
  assets,
  portfolio,
  risks
}

Ensure:
- Clean schema
- No hallucinated fields
